# Scripts de mantenimiento

Herramientas opcionales para trabajo local. **No forman parte del sitio publicado.**

| Archivo | Uso |
|---------|-----|
| `fetch-cobertura.js` | Descarga / actualiza el GeoJSON de cobertura |
| `fix-whatsapp-encoding.py` | Revisa encoding de mensajes precargados de WhatsApp |

Ejemplos:

```bash
node scripts/fetch-cobertura.js
python scripts/fix-whatsapp-encoding.py
```

Requisitos según script (Node.js o Python 3).
